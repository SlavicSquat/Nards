Эта программа является электронной версией коротких нард.
Рассчитана на двух игроков, поэтому не имеет искусственного интеллекта.
Ходы не запрограммированны, то есть шашки можно помещать куда угодно.
Чтобы переместить шашку достаточно кликнуть на неё; чтобы отпустить шашку, нужно так же кликнуть по той же шашке.
Чтобы "выбить" шашку нужно нажать "стрелку вверх".
Бросать кости можно сколько угодно раз, при чём даже во время поднятой шашки.
Правила можно посмотреть во вкладке "Правила", но если их не будет по каким-либо причинам вы можете перейти сюда https://www.10kor.ru/faq/igry-igrushki/pravila-igry-nardy для ознакомления с правилами игры

Веселья и побед вам!
